# Arrangements

Short answer: an arrangement is a playable part inside a feedpak, such as lead guitar, rhythm guitar, bass, drums, keys, or vocals.

## What An Arrangement Contains

An arrangement can include timing and note/chord data for one part. Fretted arrangements also include string and fret information.

## User-Facing Examples

- Lead guitar.
- Rhythm guitar.
- Bass.
- Drums.
- Keys.
- Vocals.

## Why Arrangement Choice Matters

The selected arrangement controls:

- What notes appear.
- Which tuning is expected.
- Which plugin views make sense.
- What scoring compares against.

## Common Problems

| Problem | Try This |
|---|---|
| Wrong notes appear | Choose a different arrangement. |
| Tuning prompt looks unexpected | Confirm the selected arrangement. |
| Plugin view is missing | The arrangement may not support that plugin. |
