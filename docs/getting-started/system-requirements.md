# System Requirements

Short answer: FeedBack Desktop needs a supported desktop operating system, audio output, and an audio input if you want tuning, calibration, or scoring.

## Supported Platforms

| Platform | Notes |
|---|---|
| Windows | Windows 10/11 x64 is the expected desktop target. |
| macOS | macOS desktop builds are expected for supported Mac hardware. |
| Linux | AppImage or Debian/Ubuntu-style packages may be available depending on release. |
| Web/server | Advanced or hosted setups may differ from the desktop app. |

## Recommended Hardware

- Guitar, bass, keys, drums, or another supported instrument path.
- Headphones or speakers.
- Audio input for tuning/scoring: microphone, USB instrument cable, or audio interface.
- Stable internet connection for downloads, updates, plugin installation, and model downloads where needed.

## Audio Input Recommendations

| Setup | Best For | Notes |
|---|---|---|
| USB instrument cable | Simple guitar/bass setup | Direct signal, usually easier than a room mic. |
| Audio interface | Lowest-friction serious setup | Pick the correct driver and channel. |
| Microphone | Quick start | Can pick up room noise and backing audio. |
| Virtual device | Routing workflows | More complex; verify the actual signal path. |

## Advanced Feature Requirements

| Feature | Extra Requirements |
|---|---|
| Stem separation | May require Demucs server and a GPU for practical speed. |
| Lyrics alignment | May require backend model downloads. |
| Low-latency desktop audio | Audio interface and proper driver path recommended. |
| Streaming routes | Desktop app and platform-specific audio routing. |
