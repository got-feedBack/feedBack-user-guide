# Getting Songs

Short answer: FeedBack plays compatible song files. Only use songs you have the right to use, and follow the import path supported by your build.

## Sources

FeedBack can work with song packages and authored content. Depending on your setup and plugins, you may be able to:

- Import a feedpak file.
- Add files to the songs folder and scan.
- Create or edit your own song package.
- Use a plugin to help import or author a chart.

## Legal and Safety Notes

- Do not upload, share, or distribute copyrighted audio or charts unless you have the right to do so.
- Keep personal backups of files you edit.
- Treat community files as untrusted until you know where they came from.
- Use the official release and plugin sources where possible.

## Import Paths

| Path | Use When |
|---|---|
| Import file | You have a feedpak somewhere on disk. |
| Songs folder + scan | You manage a local song folder. |
| Author/edit | You are building your own song package. |
| Find More or import plugins | You are discovering or preparing content through a plugin. |
