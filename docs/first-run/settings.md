# Settings Overview

Short answer: Settings is where you manage audio devices, plugins, diagnostics, updates, and app preferences.

## Common Settings Areas

| Area | Use It For |
|---|---|
| Audio | Input device, output device, channels, driver path. |
| Instrument | Instrument type, string count, tuning, reference pitch where supported. |
| Plugins | Enable, disable, configure, install, or update plugins. |
| Library | Songs folder, scan/rescan, metadata behavior where supported. |
| Diagnostics | Export logs and environment details for support. |
| Appearance | Themes or display preferences where supported. |

## First Things To Check

1. Audio input.
2. Instrument setup.
3. Calibration.
4. Plugin status.
5. Diagnostics export location.

## Troubleshooting From Settings

- If tuning or scoring fails, check Audio first.
- If a plugin button is missing, check Plugins.
- If a song is missing, check Library and scan/rescan.
- If reporting a bug, export diagnostics after reproducing the issue.
