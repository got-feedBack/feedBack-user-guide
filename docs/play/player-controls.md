# Player Controls

Short answer: player controls start, stop, pause, seek, and manage the current song.

## Core Controls

| Control | Meaning |
|---|---|
| Play | Start or resume playback. |
| Pause | Stop playback temporarily. |
| Seek | Move to another time in the song. |
| Restart | Return to the beginning or selected section where supported. |
| Speed | Slow down or speed up practice playback where supported. |
| Loop | Repeat a selected section or range where supported. |

## Before Playing

1. Confirm the arrangement.
2. Tune if needed.
3. Confirm input if scoring.
4. Choose any plugin views you want open.

## During Playback

- Use pause when tuning, changing input, or reviewing a phrase.
- Use speed and loop for hard sections.
- Avoid changing too many plugin views while diagnosing playback problems.
